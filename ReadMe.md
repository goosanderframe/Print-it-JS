# Site Web Print-It 

## Fonctionnalité

- Navigation avec flèches gauche / droite
- Défilement des slides dans le bon ordre
- Affichage de bullet points dynamiques
- Mise à jour du bullet actif
- Changement du texte associé à chaque slide
- Boucle infinie du carrousel



## Objectif
   
Ce projet a été réalisé dans le cadre de la formation OpenClassrooms afin de :

- Manipuler le DOM avec JavaScript
- Gérer les événements utilisateur
- Créer un carrousel dynamique
- Appliquer les fondamentaux du JavaScript


## Technologies utilisées (HTML, CSS, JS)
   
- HTML
- CSS
- JS Vanilla
  
## Installation

Cloner le projet :
```bash git clone https://github.com/goosanderframe/Print-it-JS.git```
